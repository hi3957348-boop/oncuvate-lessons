# 🧠 마음 탐험대 — 어린이 인지왜곡 그룹수업 16회차 기획서

> 원자료: 「인지왜곡이 뭔데?」 (쑥쑥쌤, blog.naver.com/emmatutu)
> 제작 방식: 파이어베이스(Realtime Database) 연동 · 교사–학생 실시간 상호작용 HTML 수업자료
> (기존 `thirsty.html` / `roma_summer.html` 세션 동기화 구조 재사용)

## 1. 기획 개요

### 대상 · 형태 (기본 가정 — 조정 가능)
| 항목 | 설정 |
|---|---|
| 대상 | 초등 3~6학년 (읽기·자기표현 가능 연령) |
| 형태 | 소그룹 4~8명 그룹수업 |
| 1회차 시간 | 40~50분 |
| 총 회차 | 16회차 (주 1회 기준 약 4개월) |
| 진행 | 교사 1명 + 학생 태블릿/노트북 (세션 코드로 동시 접속) |

### 회차 구성 원칙
- **도입 2회차 + 유형 학습 12회차 + 통합·마무리 2회차 = 16회차**
- 12가지 인지왜곡을 **한 유형당 정확히 한 회차씩** 충분히 다룬다. (묶음 없음 → 각 유형에 온전한 발견·도전·재구성 시간 확보)
- 별도 **도입 회차**로 세계관·개념·자동적 사고를 탄탄히 깔고, 별도 **마무리 회차**로 배운 걸 통합·발표·수료한다.

### 핵심 컨셉 — "생각 도둑을 잡아라!"
원자료의 **드라큘라 캐릭터**("너를 럭키비키로 만들어주마")를 계승해,
인지왜곡을 **마음에 몰래 들어와 진실을 왜곡하는 "생각 도둑(Thought Thief)"** 캐릭터로 의인화한다.
- 12가지 인지왜곡 = 12명의 개성 있는 생각 도둑 캐릭터 (한 회차에 한 명씩 만남)
- 아이들은 매 회차 새로운 생각 도둑을 만나 → **정체를 알아채고(발견)** → **증거로 반박하고(도전)** → **균형 잡힌 생각으로 물리치는(재구성)** 탐정이 된다.
- 거울 비유(깨진 거울·오목·볼록거울에 비친 왜곡된 나) = "생각 도둑의 마법 거울"로 시각화.

### 교육 목표 (전체)
1. 상황–**자동적 사고**–감정–행동의 연결 고리를 이해한다.
2. 12가지 인지왜곡 유형을 자기 언어로 알아챈다.
3. 왜곡된 생각에 **증거로 질문(인지 재구성)** 하는 습관을 기른다.
4. 나와 친구의 마음을 존중하며 **긍정적·균형적 사고**를 연습한다.

## 3. 매 회차 공통 수업 흐름 (템플릿)

각 회차 HTML 자료는 아래 **5단계 페이지 흐름**을 공유한다. (기존 자료의 페이지 네비게이션 + 세션 배지 구조 그대로)

```
[1] 표지          — 회차 제목 · 오늘 만날 생각 도둑 소개 (캐릭터 등장 애니메이션)
[2] 마음 열기      — 지난 회차 복습 + 오늘의 이야기 상황 제시 (교사 낭독/투표)
[3] 도둑 정체 밝히기 — 인지왜곡 개념 설명 + 예시 (원자료 문장 활용)
[4] 상호작용 활동   — 실시간 참여형 미션 (핵심, 파이어베이스 동기화)
[5] 마음 정리      — 오늘 배운 것 + 실천 미션 카드 + 감정 체크
```

### 파이어베이스 실시간 연동 설계 (기술 공통 규격)
기존 `sessions/{code}/{lesson}/{key}` 구조를 그대로 확장한다.

| 동기화 키 | 용도 (모든 회차 공통 재사용) |
|---|---|
| `page` | 교사가 넘기면 전원 페이지 이동 (수업 진도 통제) |
| `poll` | 실시간 투표/선택 결과 집계 (막대그래프 갱신) |
| `board` | 학생들이 올린 문장·생각이 실시간으로 벽에 붙는 공유 보드 |
| `match` | 드래그 매칭(도둑↔대사, 상황↔감정) 상태 공유 |
| `canvas` | 감정 그림/표정 그리기 공유 캔버스 |
| `detective` | 왜곡 생각 → 균형 생각 변환 워크시트(짝 활동) |
| `reaction` | 이모지 반응(👍❤️😮) 실시간 표시 — 서로 응원 |

- **역할 분기**: 입장 시 `강사 / 학생` 선택(기존 모달 재사용). 교사만 `page`·정답 공개 권한, 학생은 입력·투표.
- **익명 보호**: 민감한 감정 표현은 닉네임/색깔카드로 익명 게시 가능(안전한 자기개방).
- **재사용 컴포넌트**: 위 7개 위젯을 공용 모듈로 만들어 16개 파일이 조립만 하도록 설계 → 제작 효율 ↑.

## 4. 회차별 상세 설계

### 🧭 [도입 ①] 1회차 — 마음 탐험대 출발! (인지왜곡이란?)
- **학습목표**: "생각의 오류"라는 개념을 마법 거울 비유로 이해하고, 수업 세계관·약속을 세운다.
- **원자료 활용**: p.1~3 (드라큘라, "생각의 오류", 거울 비유)
- **흐름**
  - 도입: 드라큘라 → "생각 도둑" 세계관 소개. 탐험대 이름·규칙(비밀 지키기·서로 존중) 정하기.
  - 전개: **마법 거울 체험** — 같은 상황 사진 한 장을 오목/볼록/깨진 거울처럼 서로 다르게 해석해본다.
  - 개념: "인지왜곡 = 우리 머릿속에서 일어나는 **생각의 오류**. 상황을 잘못 이해해 다르게 느끼는 것."
- **상호작용 활동 🔥 [실시간 투표 `poll` + 공유 보드 `board`]**
  - 애매한 그림(친구가 나를 안 보고 지나감)을 제시 → "이 상황, 어떻게 느껴질까?" 해석 선택/입력 → 막대그래프로 함께 보며 **"같은 일도 사람마다 다르게 본다"** 체감.
  - "내 마음에도 생각 도둑이 있었던 순간" 한 줄 익명 게시.
- **실천미션**: "이번 주, 내 기분이 확 바뀐 순간 1개 관찰해 오기" 카드.

### ⚡ [도입 ②] 2회차 — 번개 생각의 비밀 (자동적 사고)
- **학습목표**: 상황→**자동으로 떠오른 생각**→기분→행동의 연결을 안다.
- **원자료 활용**: p.16 (자동적 사고, "긍정적인 생각을 갖도록 노력")
- **흐름**: 전구+톱니 아이콘 → "번개처럼 순식간에 떠오르는 생각"이 기분을 좌우함을 설명.
- **상호작용 활동 🔥 [생각 사슬 `match` + `board`]**
  - 교사가 상황 카드("발표에서 실수했다") 제시 → 학생이 *그때 번개처럼 떠오른 생각* 입력 → 그 생각이 부르는 *감정* 이모지 연결 → "상황–생각–감정" 사슬이 실시간 완성.
  - 같은 상황인데 사람마다 다른 생각→다른 감정이 나온다는 걸 나란히 비교.
- **정리**: "생각을 바꾸면 기분도 바뀐다" — 앞으로 만날 12명의 생각 도둑 예고.

### ⬛⬜ [유형] 3회차 — 흑백 도둑 (흑백논리)
- **원자료 활용**: p.4 (좋다/나쁘다, "나는 완전히 실패자야")
- **개념**: 모든 걸 100 아니면 0으로 나누는 도둑.
- **상호작용 활동 🔥 [슬라이더 스펙트럼 `poll`/`detective`]**
  - **회색지대 찾기**: "시험 한 문제 틀림 = 완전 실패?" 문장을 **0~100 감정 슬라이더**로 실제 정도 표시 → "0과 100 사이엔 회색이 많다" 시각화.
  - 흑백 문장("난 항상 못해") → 균형 문장("이번엔 아쉬웠지만 잘한 것도 있어")으로 짝이 바꿔 쓰기.

### 🔁 [유형] 4회차 — 항상·결코 도둑 (과잉 일반화)
- **원자료 활용**: p.11 ("한 번의 경험으로 모든 것 판단", '항상'·'결코')
- **상호작용 활동 🔥 [단어 사냥 `board` + `reaction`]**
  - **'항상·결코' 단어 사냥꾼**: 예시 문장 속 과장 단어(항상/절대/맨날/결코)를 찾아 클릭 → 실시간 카운트, 도둑의 힘 게이지가 줄어드는 연출.
  - "한 번 넘어졌다고 **항상** 넘어지는 건 아니야" → 반례 증거를 하나씩 올려 공유 보드 채우기.

### 🎈 [유형] 5회차 — 뻥튀기 도둑 (재앙화)
- **원자료 활용**: p.12 (숙제 두고 옴 → "엄청 혼날 거야")
- **상호작용 활동 🔥 [확률 저울 `poll` + 캔버스 `canvas`]**
  - **최악 vs 실제 저울**: 걱정 상황을 주고 "정말 그 최악이 일어날 확률은?"을 슬라이더로 예측 → 현실적 결과를 이야기하며 풍선 바람 빼기 애니메이션.
  - "부풀린 문제 크기" vs "실제 크기"를 공유 캔버스에 나란히 그리기.

### 🔮 [유형] 6회차 — 독심술 도둑 (마음읽기)
- **원자료 활용**: p.6 (누가 내 얘기 안 들으면 "저 사람은 나를 재미없어해")
- **개념**: 물어보지도 않고 남의 마음을 다 안다고 착각하는 도둑.
- **상호작용 활동 🔥 [생각 vs 사실 구분 `poll` + 확인 질문 `board`]**
  - **"정말 읽은 거야, 짐작한 거야?"**: "쟤는 날 싫어해" 같은 문장이 *사실인지 내 짐작인지* 실시간 분류.
  - **확인 질문 만들기**: 마음을 넘겨짚는 대신 직접 물어볼 수 있는 말("혹시 무슨 일 있어?")을 만들어 공유.

### 🙋 [유형] 7회차 — 내탓 도둑 (내탓하기)
- **원자료 활용**: p.9 (친구 기분 안 좋으면 "내가 뭘 잘못했나?")
- **개념**: 세상 모든 일의 원인을 나에게 돌리는 도둑.
- **상호작용 활동 🔥 [책임 파이차트 `poll`/`canvas` + `board`]**
  - **"내 몫은 얼마나?"**: "친구가 시무룩하다" 상황의 여러 원인(피곤함/가정 일/시험 등)을 함께 찾아 **파이차트로 원인 나누기** → 내 탓은 전체 중 일부임을 시각화.
  - 다른 사람·상황·우연의 몫을 학생들이 입력해 원 그래프를 채운다.

### ☁️ [유형] 8회차 — 먹구름 도둑 (부정적 선택)
- **원자료 활용**: p.10 (좋은 일 많았는데 나쁜 일 하나로 하루 망침)
- **개념**: 좋은 건 안 보이고 나쁜 것만 걸러 보는 필터 도둑.
- **상호작용 활동 🔥 [오늘의 좋은 일 수집기 `board` + `poll`]**
  - 학생 각자 오늘 있었던 좋은 일 3가지를 올려 공유 벽 채우기 → "좋은 일이 이렇게 많았네!"
  - 나쁜 일 하나 vs 좋은 일 여러 개를 저울에 올려 "하루 전체" 균형 있게 보기.

### 🔻 [유형] 9회차 — 깎아내리기 도둑 (작게보기)
- **원자료 활용**: p.7 (칭찬받으면 "그냥 운이 좋아서")
- **개념**: 좋은 일·내 노력을 일부러 작게 만드는 도둑.
- **상호작용 활동 🔥 [칭찬 되돌려주기 `detective` + `reaction`]**
  - "그냥 운이 좋았어" 대신 *내가 한 노력·실력*을 찾아 말하기 짝 활동.
  - 친구의 좋은 점을 찾아 **칭찬 카드**를 서로에게 보내고, 받은 칭찬을 "운"으로 깎지 않고 그대로 받기 연습.

### 🔥 [유형] 10회차 — 채찍 도둑 (엄격한 평가)
- **원자료 활용**: p.8 (무척 멋진 그림에도 "이건 별로야")
- **개념**: 나에게만 유독 가혹한 잣대를 들이대는 도둑.
- **상호작용 활동 🔥 [친구 목소리로 바꾸기 `detective` + 잘한 점 찾기 `board`]**
  - **"친구에게라면 뭐라고 할까?"**: 나를 향한 가혹한 말 입력 → *똑같은 상황의 친한 친구에게 해줄 말*로 바꿔 쓰기 → 화면에 나란히 대비.
  - 내 결과물에서 **잘한 점 3가지** 찾아 올리기(스스로 인정 연습).

### 💯 [유형] 11회차 — 완벽 도둑 (완벽주의)
- **원자료 활용**: p.15 (글씨 조금 틀리면 "이건 엉망이야")
- **개념**: 100%가 아니면 전부 실패라고 몰아붙이는 도둑. (흑백논리와 형제지간 — 짧게 연결 복습)
- **상호작용 활동 🔥 ["충분히 좋음" 기준 만들기 `poll`/`detective` + `reaction`]**
  - **실수 괜찮아 스탬프**: 작은 실수가 있는 결과물도 충분히 좋은 예를 함께 보고 "이 정도면 충분해!" 기준선을 슬라이더로 정하기.
  - "완벽하게" 대신 "즐겁게·끝까지" 같은 새 목표 문장으로 바꿔 게시.

### ⚖️ [유형] 12회차 — 비교 도둑 (비교하기)
- **원자료 활용**: p.14 (남과 비교해 부족하다, "모두가 다른 장점이 있다")
- **상호작용 활동 🔥 [나의 반짝임 지도 `canvas`/`board`]**
  - **강점 별자리**: 학생마다 "나만 가진 좋은 점"을 별로 올려 반 전체가 서로 다른 별자리를 이룸 → "모두 다른 장점이 있다" 시각화.
  - 비교 문장("쟤보다 내가 못해") → "우리는 각자 잘하는 게 달라"로 바꾸기.

### 👀 [유형] 13회차 — 눈치 도둑 (남의 기준으로 판단하기)
- **원자료 활용**: p.13 (친구가 새 옷을 칭찬 안 하면 그 옷이 별로)
- **개념**: 나의 가치를 남의 반응에 맡기는 도둑.
- **상호작용 활동 🔥 [내 기준 vs 남 기준 분류 `match` + `board`]**
  - "내가 좋아서 고른 것" vs "남 눈치 봐서 정한 것"을 분류 → 내 기준으로 정하는 연습.
  - "남이 칭찬 안 해도 내가 좋으면 좋은 것" 나만의 기준 문장 만들어 게시.

### 🪞 [유형] 14회차 — 과잉반사 도둑 (과잉반사)
- **원자료 활용**: p.5 (작은 일을 크게, 친구가 바빠서 못 본 걸 "나를 싫어해")
- **개념**: 작은 신호를 나에게 향한 큰 의미로 되튕겨 받는 도둑.
- **상호작용 활동 🔥 [신호 크기 맞추기 `poll` + `detective`]**
  - 작은 상황(친구가 톡 답이 늦음)을 주고 "이 신호의 진짜 크기는?" 실제 크기 맞히기 → 부풀린 해석과 현실 대비.
  - 지금까지 만난 도둑들과 연결: "이건 어떤 도둑과 닮았지?" 살짝 복습으로 15회차 연결.

### 🕵️ [통합] 15회차 — 생각 탐정 훈련소 (인지 재구성)
- **학습목표**: 왜곡 생각에 스스로 질문하고 **균형 잡힌 생각**으로 바꾼다. 12유형 종합 복습.
- **상호작용 활동 🔥 [도둑 도감 매칭 `match` + 4단계 탐정 워크시트 `detective`]**
  - **생각 도둑 도감 완성 & 정체 맞히기 퀴즈쇼**: 12명의 도둑 ↔ 대표 대사 실시간 매칭 팀전, 상황+생각 보고 "어떤 도둑?" 버저 투표.
  - **탐정의 5가지 질문** 카드: ①진짜 증거 있어? ②반대 증거는? ③친구라면 뭐라 할까? ④다르게 볼 순 없을까? ⑤최악 말고 현실은?
  - 자기 고민 상황을 넣어 *번개생각 → 탐정질문 → 균형생각*으로 변환, 짝과 실시간 공유·피드백.

### 🎉 [마무리] 16회차 — 마음 튼튼 발표회 (마무리 · 수료)
- **상호작용 활동 🔥 [나만의 긍정 생각 카드 `canvas`/`board` + `reaction`]**
  - **나만의 긍정 생각 카드** 제작: 자주 찾아오는 내 생각 도둑 1명 + 그를 물리칠 나만의 균형 문장을 카드로 디자인 → 공유 갤러리 전시.
  - **마음 탐험대 수료식**: 서로의 카드에 응원 이모지, 수료증 발급. "긍정적인 생각을 갖도록 노력하는 게 중요"(원자료 p.16)로 마무리.

## 6. 다음 단계 제안 (제작 로드맵)

1. **공용 엔진 파일** 먼저 제작: 파이어베이스 초기화 + 7개 재사용 위젯(`poll`, `board`, `match`, `canvas`, `detective`, `reaction`, `page`) 모듈화.
2. **1회차 파일**을 레퍼런스로 완성 → 세계관·디자인 톤·상호작용 검증.
3. 검증된 템플릿을 복제해 2~16회차 콘텐츠만 채워 넣기.
4. 생각 도둑 12캐릭터 일러스트/이모지 세트 확정(원자료 드라큘라 톤 유지).

> - 실제 대상 학년·그룹 인원·1회차 길이

> - 캐릭터 일러스트를 새로 그릴지, 이모지/도형 기반으로 갈지
